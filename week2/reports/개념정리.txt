===== RBF 커널 (Radial Basis Function) =====

1. 기본 개념
   일반 SVM은 데이터를 직선(선형)으로 나눈다.
   하지만 현실 데이터는 직선으로 깔끔하게 나뉘지 않는 경우가 많다.
   RBF 커널은 데이터를 더 높은 차원으로 변환하여,
   원래 공간에서는 직선으로 나눌 수 없던 것을 곡선 경계로 나눌 수 있게 한다.

2. 직관적 비유
   2D 평면에 정상/이상 데이터가 섞여 있을 때:
   - 선형 SVM: 직선 하나로 나눔
   - RBF SVM: 데이터를 3D로 들어올린 뒤 평면으로 자름
     → 원래 2D에서 보면 원형/곡선 경계가 됨

3. 수식
   K(x, y) = exp(-γ × ||x - y||²)

   - 두 데이터 포인트 간 거리를 계산
   - 가까운 점은 유사도 높음(≈1), 먼 점은 유사도 낮음(≈0)
   - γ(감마): 영향 범위 조절
     γ가 크면 가까운 점만 고려 → 복잡한 경계
     γ가 작으면 멀리까지 고려 → 부드러운 경계

4. 프로젝트에서의 의미
   VIX/GPR 같은 변수들 사이의 복잡한 관계(단순 비례가 아닌)를
   포착할 수 있다는 것이 RBF 커널을 선택한 이유이다.


===== StandardScaler (표준화) =====

1. 기본 개념
   데이터의 각 Feature를 평균 0, 표준편차 1로 변환하는 정규화 방법이다.

2. 수식
   Z = (X - μ) / σ
   - X: 원래 값
   - μ: 평균
   - σ: 표준편차

3. 왜 필요한가
   예를 들어 이 프로젝트에서:
   - VIX: 10~80 범위
   - GPR: 0~500 범위
   - 수익률: -10~+10% 범위

   스케일이 다르면 SVM, KNN, Logistic Regression 같은 모델은
   숫자가 큰 Feature에 끌려간다.
   StandardScaler로 모두 같은 스케일로 맞춰야 공평하게 학습된다.

4. 트리 기반 모델은 불필요
   Random Forest, XGBoost 등 트리 기반 모델은
   값의 크기 순서만 보고 분할하기 때문에 스케일 영향을 받지 않는다.
   따라서 SVM, KNN, Logistic Regression 등
   스케일에 민감한 모델에만 적용하였다.
